# bitcoin_blockChain
Implementation of Block Chain protocol in a Simple_Bitcoin in nodejs

----------


How to run
-------------


> **Commands to run (run each command in different terminal):**

> - **git@github.com:cs12b015/bitcoin_blockChain.git**
> - **cd bitcoin_blockChain**
> - **node app.js **.
> - **node client.js 1**. 
> - **node client.js 2**. 
> - **node client.js 3**. 
> - **node client.js 4**. 
> - **node client.js 5**.
